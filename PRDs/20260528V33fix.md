# V3.3：小白版/专业版口径统一与观察池分层

当前问题不是单点 bug，而是报告口径不统一。请不要继续零散修一个指标，而是做一次统一口径修复。

## 一、统一市场指标命名

现在小白版“市场情绪评分”用 sentiment，专业版“市场情绪评分”用 market，导致同一天小白版 55.1 / 平衡，专业版 61.0 / 偏强。

修改：
- market.score/status 改名展示为“市场宽度评分/市场宽度状态”
- sentiment.score/stage 改名展示为“短线情绪周期评分/情绪周期阶段”
- 小白版和专业版都同时展示这两个指标
- 不允许两个不同指标都叫“市场情绪评分”

## 二、修复 market.py 评分偏强问题

2026-05-27：
- 上涨 974
- 下跌 4489
- 涨停 64
- 跌停 40
- 成交额 32599 亿

不能显示“市场整体偏强”。应显示“分化”或“宽度偏弱、局部热点活跃”。

## 三、统一板块名称归一

对以下位置统一使用 normalize_board_name：
- render_board_table()
- render_ratio_change_table()
- 今日风险方向
- 今日主线判断
- 板块联动命中标签

避免专业版继续出现：
- 白酒Ⅱ / 白酒Ⅲ
- 证券Ⅱ / 证券Ⅲ
- IT服务Ⅱ / IT服务Ⅲ

## 四、解释“今日主线”和“板块趋势摘要”的区别

在报告中明确：
- 今日主线判断：当前综合强度
- 板块资金趋势摘要：生命周期变化
二者不完全等同。

## 五、主线文案分类

theme_detector.py 增加：
- 上涨主线
- 分歧主线
- 回流主线

如果 pct_chg < 0 但成交占比上升，不要写“涨幅靠前”，应写“资金关注度提升但板块当日分歧较大”。

## 六、观察池分层

不要把所有筛选结果都叫“观察池”。

分成：
- 可观察池：低/中风险，非回避
- 谨慎观察池：中高风险
- 高风险复盘池：高风险、回避、数据不足、异常涨幅、新股

小白版默认优先展示可观察池，回避票不要放在首屏普通观察池。

## 七、修复入选逻辑

1. 短线强势：
   - 排除 N 开头新股
   - 要求 volume_ratio、ma5、ma10 有效
   - 涨幅异常过高的票进入高风险复盘池，不进普通观察池

2. N字异动 / 二次起爆：
   - 增加 pct_20d <= 50 或 <= 60
   - 超过上限放入高位强势复盘池

3. 板块联动：
   - normalize_board_name
   - 过滤 LOW_VALUE_BOARDS
   - 不用富时罗素、创业板综、深股通、昨日高振幅作为强势板块理由

## 八、数据质量拆分

报告可信度不再只给一个 80 分。

拆成：
- 行情数据可信度
- 板块数据可信度
- 技术指标可信度
- 观察池可信度

如果 79% 个股缺少均线，则技术指标可信度应明显偏低，并提示依赖均线的选股结果仅供参考。

## 九、专业版结构修复

专业版标题必须在最前面，不要先输出数据质量检查再输出标题。

## 十、验收

用 2026-05-27 数据重新生成：

1. 小白版和专业版不再出现同名不同分数的“市场情绪评分”；
2. 市场不再显示“整体偏强”；
3. 专业版行业榜单不再出现白酒Ⅱ/白酒Ⅲ等重复层级；
4. 半导体下跌时不再写“涨幅靠前”；
5. 短线强势池不再出现 N长进、N龙辰；
6. N字/二次起爆不再出现 20日涨幅超过 60% 的普通观察标的；
7. 板块联动不再用富时罗素/深股通/昨日高振幅作为核心命中理由；
8. 专业版标题在文件第一行；
9. 高风险/回避票和普通观察票分开展示。

## 第一轮 V3.3-A：报告口径统一与明显错误修复（本轮执行）

### 1. 市场指标命名统一
- `market.py`：`score` → "市场宽度评分"，`status` → "市场宽度状态"
- `sentiment.py`：`score` → "短线情绪周期评分"，`stage` → "情绪周期阶段"
- `report_renderer.py`：小白版和专业版都展示两个指标，不允许同名不同分

### 2. market.py 评分和状态修复
- 新增"高度分化 / 宽度偏弱、局部热点活跃"状态
- 规则：上涨占比 < 30% 且涨停数 ≥ 50 → "分化"
- 上涨占比 < 50% 且下跌 > 上涨 → "宽度偏弱"
- 修改评分公式权重，today_ratio 降权

### 3. 板块名称归一全覆盖
- `render_board_table()`：board_name → normalize_board_name()
- `render_ratio_change_table()`：同理
- 今日风险方向：同理
- 今日主线判断：同理（theme_detector.py 已有，确认生效）
- 验收：专业版不再出现 白酒Ⅱ/Ⅲ、证券Ⅱ/Ⅲ

### 4. 主线 vs 趋势摘要增加说明
- 小白版/专业版都加一行说明：
  "今日主线判断 = 当前综合强度（涨幅+成交占比+领涨股）"
  "板块资金趋势摘要 = 资金趋势和生命周期变化，二者不完全等同"

### 5. 主线三分
- `theme_detector.py`：新增 theme_type 字段
  上涨主线：pct_chg > 0 且强度高
  分歧主线：pct_chg < 0 但成交占比上升 → "资金关注度提升但板块分歧较大"
  回流主线：前期退潮后资金回流
- 文案中不再对所有主线使用"涨幅靠前"

### 6. 专业版标题前置
- 第一行：`# A股日报 · 专业版 | YYYY-MM-DD`
- 第二段：数据获取概要（股票/行业/概念数量）
- 第三段：数据质量检查

### 7. 日报趋势摘要加图例
- 小白版/专业版趋势摘要下方加一行小字：
  "来源：板块资金趋势追踪 | 优先参考近5日资金变化和生命周期迁移"
