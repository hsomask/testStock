## 5. 修复 board_amount_ratio 的成交额来源

不要完全依赖 stock_board_industry_name_em / stock_board_concept_name_em 是否返回“成交额”字段。

更稳的逻辑：
- 获取全市场个股行情 stock_df；
- 获取每个板块成分股；
- 按成分股 code 匹配 stock_df 的 amount；
- 汇总得到板块成交额；
- amount_ratio = 板块成交额 / 全市场成交额。

如果接口暂时不可用，则报告显示“板块成交额暂缺”，不要生成错误的成交占比。
